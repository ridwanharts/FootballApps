## Feature Project

* Menggunakan API themoviedb (https://www.themoviedb.org/)

* Library Picasso untuk menampilkan gambar
* Retrofit 2 untuk komunikasi dengan API
* Fitur share movie
* Fitur Movie Now Playing, Upcoming, Favourite
